Desafio 1

Crie uma página HTML contendo:

  • Título principal

    • Conteúdo do título o seu nome
    • O título deve estar centralizado na página

  • Uma parágrafo

    • Conteúdo deve ser um textinho dizendo "quem é você na fila do pão" (um resumo rápido de quem é você como se você estivesse se apresentando para um estranho)
    • O texto deve estar alinhado a esquerda

  • Um subtítulo

    • Conteúdo deve ser "profissão"
    • Este título deve estar alinhado a esquerda

  • Uma parágrafo

    • Conteúdo deve ser um resumo sobre a sua profissão
    • O texto deve estar justificado
    • Deve conter pelo menos 1 nome de alguém que trabalha com você e o nome dessa pessoa deve estar em negrito

  • Um subtítulo

    • Conteúdo deve ser sua "família"
    • Este título deve estar alinhado a esquerda

  • Uma parágrafo

    • Conteúdo deve ser um resumo sobre a sua família
    • O texto deve estar alinhado a direita
    • Deve conter pelo menos 1 nome de alguém da sua família e este nome deve estar em negrito
